A NLP-Based Approach To Datepicker Identification Using BERT & Binary Classification Neural Networks

The Problem

Automate the identification of date picker trigger elements & container elements

The Solution

Pre-train a BERT model network with HTML represented in Emmet syntax, and leverage the resultant network to fine tune a model to identify HTML as a datepicker trigger or container element as a binary classification task

What is BERT?

  • Open-sourced neural network-based technique for NLP released by Google
  • Processes words in relation to all other words in a sentence rather than one-by-one order

Why BERT?

  • Traditional machine learning approaches require training against labeled data
  • BERT is able to understand human language in context without labeled data
  • HTML shares similarities to human language
  • BERT offers an approach to creating a model that understands HTML
